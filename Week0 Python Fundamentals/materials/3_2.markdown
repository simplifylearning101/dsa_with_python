# Hour 3 Homework: Practice Problems

Practice these 10 programs to master functions. Write each as a .py file and verify outputs.

1. **Problem**: Write a function to print a greeting.  
   **Outcome**: Print "Hello, [name]!".  
   **Example**: name="Alice" → Output: "Hello, Alice!"

2. **Problem**: Function to return square of a number.  
   **Outcome**: Return n * n.  
   **Example**: n=4 → Output: 16

3. **Problem**: Function to check if number is positive.  
   **Outcome**: Return True/False.  
   **Example**: num=5 → Output: True

4. **Problem**: Function to find maximum of two numbers.  
   **Outcome**: Return larger number.  
   **Example**: a=3, b=7 → Output: 7

5. **Problem**: Function with default parameter for greeting.  
   **Outcome**: Print greeting with name, default "Hi".  
   **Example**: name="Bob" → Output: "Hi, Bob!"

6. **Problem**: Function to sum numbers from 1 to n.  
   **Outcome**: Return sum.  
   **Example**: n=5 → Output: 15

7. **Problem**: Function to check if number is divisible by 3.  
   **Outcome**: Return True/False.  
   **Example**: num=9 → Output: True

8. **Problem**: Function using global variable to track calls.  
   **Outcome**: Print number of times function was called.  
   **Example**: After 3 calls → Output: 3

9. **Problem**: Function to return absolute difference of two numbers.  
   **Outcome**: Return |a - b|.  
   **Example**: a=10, b=4 → Output: 6

10. **Problem**: Function to return string repeated n times.  
    **Outcome**: Return string n times.  
    **Example**: s="Hi", n=3 → Output: "HiHiHi"
